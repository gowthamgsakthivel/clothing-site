UI components for the admin design system.

- Card: rounded-2xl containers with shared padding helpers.
- Button: primary, secondary, danger, ghost; small/medium sizes.
- StatusBadge: centralized order/shipment status colors.
- Table: compact option with sticky headers and row hover.
- Pagination: simple next/prev controls.
- Modal/Drawer: overlay shells with consistent spacing.
- Skeleton/EmptyState: loading and empty UI primitives.
